#### 3.4.2 Event Loop Block检测阈值

- **主进程阻塞阈值**：500ms (影响窗口响应)
- **渲染进程ANR阈值**：5000ms (影响用户交互)
- **游戏循环阻塞阈值**：33ms (影响60fps流畅度)
- **告警升级策略**：连续3次阻塞→警告，连续10次→严重告警

#### 3.4.3 性能监控基线

- **应用启动时间**：<3秒 (从点击到主窗口显示)
- **内存占用基线**：启动<200MB，稳定运行<500MB
- **CPU占用基线**：空闲<5%，游戏运行<30%
- **磁盘I/O基线**：存档操作<100ms，资源加载<500ms
